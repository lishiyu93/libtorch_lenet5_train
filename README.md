# libtorch_lenet5_train
A libtorch training of lenet5 example.一个基于libtorch（pytorch的C++版本）平台的lenet5训练案例
## 环境
1. LibTorch v1.6.0
2. CUDA v10.1
3. OpenCV v3.4.0

5）配置OpenCV包含目录和库目录：#11
6）配置libtorch包含目录和库目录：#12
7）添加工程文件：#13
8）配置CUDA的附加依赖项：#15
